# Discord-Status-Cpu
เอาไว้โชว์การใช้งาน CPU ใน Status Discord เท่ๆ

# วิธีใช้งาน
> 1. ให้ใช้คำสั่ง **npm i, npm install, yarn install** ใน cmd ได้เลย
> 2. ต่อไปก็ให้สร้างไฟล์ .env ในไฟล์ .env ก็เขียนแบบนี้
